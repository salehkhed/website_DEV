Introduction à HTML

HTML (HyperText Markup Language) est le langage de balisage standard pour la création de pages web. Il permet de structurer le contenu en utilisant des balises et des éléments. Chaque balise a un rôle spécifique dans la définition de la structure du document.

Éléments HTML

Les éléments HTML sont les blocs de construction de base d'une page web. Chaque élément est entouré de balises, et les balises définissent la signification et la fonction de chaque élément.

  • Balise <html> : Définit le début et la fin du document HTML.
  • Balise <head> : Contient des métadonnées, des liens vers des fichiers CSS, des scripts, etc.
  • Balise <body> : Contient le contenu principal de la page.
  • Balise <h1> à <h6> : Définit les titres de différentes tailles.
  • Balise <p> : Définit un paragraphe de texte.
  • Balise <a> : Crée un lien hypertexte.

Attributs HTML

Les attributs fournissent des informations supplémentaires sur les éléments HTML. Chaque attribut a un nom et une valeur, et ils sont ajoutés aux balises pour modifier leur comportement.

  • Attribut `src` : Spécifie la source (URL) d'une image ou d'une ressource externe.
  • Attribut `href` : Spécifie l'URL vers laquelle le lien pointe.
  • Attribut `alt` : Fournit un texte alternatif pour les images (utile pour l'accessibilité).
  • Attribut `class` : Permet de donner une classe à un élément, utile pour le stylage CSS.
  • Attribut `id` : Fournit un identifiant unique à un élément.
